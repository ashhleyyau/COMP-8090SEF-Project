# COMP-8090SEF-Project
Task 1: Supermarket Inventory System
A python-based system with GUI(Tkinter) demonstration.

## Features
- Add, restock, sell and delete products
- Three product types: Fresh Food, Groceries, Household
- Automatic product numbering
- Records for each restock/sell operation

## How to Run
1. Clone the repository
2. Navigate to the Task 1 folder
3. Run `python main.py`

## File Structure
- `main.py` – Program entry point
- `gui.py` – Tkinter GUI implementation
- `product.py` – Product class hierarchy
- `inventory.py` – Inventory and record management
